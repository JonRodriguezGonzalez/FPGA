# FPGA
This repository will include personal proyects involving FPGAs, mainly for self-learning purposes. Until a full FPGA is purchased all code is simulated using the LABSLAND-FPGA virtual laboratory, where you can access real FPGAs and use them in real time for a small suscription. For more information, visit LABSLAND.

We will be making use of the DE-115 development board along with VHDL.
